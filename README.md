# Genetic algorithm - Find the word

The goal of this program is of generate a phrase, that you inserted, using a genetic algorithm

### Usage

```sh
$ g++ Source.cpp
$ ./a.out
```

### Example

```
Insert the phrase to generate: TrY T0 F1nD tHi5

Population: 51
Mutation: 5%

        Phrase find: TrY T0 F1nD tHi5

In 0.667351 seconds
```
